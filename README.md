## INSTALASI DATABASE

-   Buat database pada mysql terlebih dahulu dengan nama **pju**
-   Lanjut ke instalasi project

## INSTALASI PROJECT

-   clone atau download project ini
-   jika didownload (bentuk .zip) silahkan extract pada folder yang diinginkan
-   kemudian buka .env, isi DB_USERNAME=root dan DB_PASSWORD= (dikosongkan)
-   kemudian ketik perintah pada terminal (dalam folder project) => php artisan migrate
-   setelah selesai jalankan project dengan perintah => php artisan serve
